My AI Project - ELIZA vs LLM

Files:
- eliza.py (ELIZA chatbot)
- LLM.py (Modern chatbot)
- chat_comparison.py (Comparison GUI)

How to run:
python eliza.py
python LLM.py
python chat_comparison.py

My test results:

Question: "Hello"
ELIZA said: Hello! How are you feeling today?
LLM said: Hello! How can I assist you today?

Question: "I feel stressed"
ELIZA said: Why do you feel stressed?
LLM said: I understand stress can be difficult.

Question: "My mother is strict"
ELIZA said: Tell me about your relationship with your mother.
LLM said: Could you tell me more about that?

What I learned:
- ELIZA uses pattern matching (looks for keywords)
- LLM tries to understand context
- LLM gives better responses than ELIZA
